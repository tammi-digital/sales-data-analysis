# Sales Data Analysis

This project analyzes sales data using Python.

The script reads a CSV dataset and calculates:

- Total sales
- Average sales
- Highest sales day
- Lowest sales day

Example output:

Total Sales: 1640
Average Sales: 234
Best Day: Thursday 420
Worst Day: Wednesday 90

Skills demonstrated:

- Python
- CSV file processing
- Data analysis
- Basic statistics
